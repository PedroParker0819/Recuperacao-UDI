Seção 3: Apresentação visual

Para apresentação visual algumas técnicas são importantes

Práticas de Texto

As seguintes práticas podem ser adotadas para garantir a acessibilidade do texto:

  • Utilize uma estrutura de cabeçalhos adequada para facilitar a navegação no conteúdo.
  • Forneça descrições alternativas para imagens e elementos visuais que transmitam informações importantes.
  • Utilize contrastes adequados entre o texto e o plano de fundo para facilitar a legibilidade.

Práticas de Imagens

As seguintes práticas podem ser adotadas para garantir a acessibilidade de imagens:

  • Utilize o atributo "alt" para fornecer descrições alternativas para as imagens.
  • Evite o uso de texto incorporado em imagens, a menos que seja essencial para a compreensão do conteúdo.
  • Verifique se as imagens possuem contraste adequado para garantir a legibilidade do conteúdo visual.

Práticas de Vídeos

As seguintes práticas podem ser adotadas para garantir a acessibilidade de vídeos:

  • Forneça legendas ou transcrições para os vídeos para permitir o acesso a pessoas com deficiência auditiva.
  • Utilize recursos de descrição de áudio para transmitir informações visuais importantes contidas nos vídeos.
  • Verifique se os controles de reprodução do vídeo são acessíveis e compatíveis com tecnologias assistivas.
